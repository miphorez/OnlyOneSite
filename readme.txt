Программа серии "Родительский контроль".
Позволяет просматривать ребенку контент по строго ограниченному URL-у.
Ребенок сам не сможет добавлять URL-ы в список, они шифруются и добавляются в адресную строку в зашифрованном виде. При добавлении сайта он попадает в список разрешенных сайтов, в последствии можно вызвать любой сайт из списка через меню.
Порядок работы:

На компьютере родителя:
1. Войти в программу в режиме "родитель": запуск программы с аргументом "-admin"
2. Найти нужную страницу, разрешенную для просмотра
3. Нажать клавиши Ctrl+Alt+Shift+H
4. Скопировать в буфер обмена
5. Передать на компьютер ребенка файл с шифрованным URL-ом

На компьютере ребенка:
1. Войти в программу
2. В адреснкю строку вставить шифрованный URL разрешенной к просмотру страницы
3. Нажать на иконку "Адрес сайта"
4. В выплывающем окне создать надпись, под которым данная страница будет храниться в списке разрешенных сайтов.

Примечание: Можно задавать как конкретный URL страницы, так и диапазон страниц. Определение разрешенного контента определяется по шаблону-строке, который определяется при формировании шифрованной записи URL. 
Например, если задать адрес: "https://www.google.com", то можно будте просматривать только эту страницу с этим доменным именем, если задать "https://www.google", то становятся доступны адреса "https://www.google.com", "https://www.google.ru", "https://www.google.by" и т.д.
